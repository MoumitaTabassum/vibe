About VIBE

The VIBE is a blocks-based programming environment with two different contexts (Grid World and Audio World). The purpose of VIBE is to provide a universal and accessible programming enviornment for both students with and without visual impairments. For each new world, the executable area appears differently. Also, there is a new palette option for each world,; e.g., the Grid World has blocks for moving the character - like “move” and “turn”. Each specific palette will only appear when its associated world is selected.
